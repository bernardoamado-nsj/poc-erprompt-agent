# POC ERPrompt Agent

POC de um sistema que usa IA para gerar telas dinamicamente.

## Setup do ambiente

### 1) Arquivo de váriaveis de ambiente
Crie um arquivo `.env` na raiz do projeto com o mesmo conteúdo do arquivo `.env.dist`.

### 2) Inicie o servidor do backend
Usando **pnpm**: 
> pnpm serve

**OU** usando **npm**:
> npm run serve

### 3) Inicie o servidor do frontend
Usando **pnpm**:
> pnpm dev

**OU** usando **npm**:
> npm run dev

O servidor rodará na porta 5199
Para abrir o sistema no navegador acesse http://localhost:5199
